# HospitalFlow

HospitalFlow is a hospital workflow modeling and improvement project that captures current processes, proposes optimized future workflows, and supports performance-oriented analysis of hospital operations. The repository brings together BPMN diagrams, UML artifacts, strategic and organizational models, dashboard files, and a Visual Paradigm project, making it a documentation- and analysis-focused system for understanding, evaluating, and redesigning admission and operational workflows in a healthcare setting.
